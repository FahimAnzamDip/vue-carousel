# Simple Vue 3 & Tailwind CSS Carousel
> ### Live URL: https://fahimanzamdip.github.io/vue-carousel/

## Local Installation
- `` git clone https://github.com/FahimAnzamDip/vue-carousel.git ``
- `` cd /path/to/project ``
- `` npm install ``
- `` npm run dev ``
